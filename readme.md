<h1> Preparation </h1>
1. composer create-project --prefer-dist laravel/laravel:^11.0 api-namaproject
2. cd api-namaproject
3. php artisan serve
4. php artisan storage:link

<H1> Database & Model </h1>
1. konfigurasi .env
2. sqlite -> mysql
3. setting database
 (name, user, password)
4. Membuat model: php artisan make:model Post -m
